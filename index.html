bindInput Example

bindInput will set an event listener on the input element. When a focus event is trigger the timepicker will be opened with the input element in context. 

var timepicker = new TimePicker();

timepicker.bindInput('#bind-example');
It is also possible to pass an HTMLElement instead of a selector. This example also uses the military time format option. 

var timepicker = new TimePicker()
    inputEl = document.getElementById('bind-example-military');

timepicker.bindInput(inputEl, {timeFormat: 'military'});

openOnInput Example

openOnInput allows for more customized usage. Instead of setting an event listener on the input element, it will open on the input passed when called. 

var timepicker = new TimePicker(),
    openOnStandard = document.getElementById('#openOn-example');

openOnStandard.addEventListener('focus', function(event) {
    timepicker.openOnInput(event.target);
});
openOnInput example using military time format option. 

var timepicker = new TimePicker(),
    openOnMilitary = document.getElementById('#openOn-example-military');

openOnMilitary.addEventListener('focus', function(event) {
    timepicker.openOnInput(event.target, {timeFormat: 'military'});
});
